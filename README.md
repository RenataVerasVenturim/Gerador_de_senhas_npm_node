# Gerador_de_senhas_npm_node
Gerador de senhas de 8 dígitos alfanumérico via npm randomstring do Node.js (backend) + servidor local

# Install

<p>npm init -y</p>
<p>npm i express</p>
<p>npm i randomstring</p>
<p>node server.js</p>
